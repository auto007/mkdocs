### **1. ShellCheck — 最强 Shell 脚本语法检查**

🔍 自动检查脚本错误、风险写法  
🛠 给出修复建议（如双引号、变量引用等）

安装名称：**ShellCheck**  
VS Code Marketplace：`timonwong.shellcheck`

---

### **2. Bash IDE — 代码补全 + 跳转 + 智能提示**

✨ 提供 Bash 语言服务器功能：

- 语法智能补全
    
- 函数跳转
    
- Go to Definition
    
- Hover 提示说明
    

安装名称：**Bash IDE**  
VS Code Marketplace：`mads-hartmann.bash-ide-vscode`

---

### **3. Bash Beautify — 格式化脚本**

🧹 可以自动格式化 *.sh 脚本，保持统一缩进和风格。

安装名称：**Bash Beautify**  
VS Code Marketplace：`shakram02.bash-beautify`

---

### **4. shell-format（可选）**

支持 sh / bash / zsh / ksh 的格式化，更强。

安装名称：**shell-format**

---

### **5. Better Shell Syntax — 语法高亮更强**

让 Shell 脚本的颜色更清晰，支持许多特殊语法。

安装名称：**Better Shell Syntax**

---

## 📦 **高质量 Shell 开发：推荐组合**

|功能|插件|
|---|---|
|语法检查|ShellCheck|
|代码补全 & 智能跳转|Bash IDE|
|格式化|shell-format 或 Bash Beautify|
|高亮|Better Shell Syntax|

📌 **这四个同时装体验最佳**。
